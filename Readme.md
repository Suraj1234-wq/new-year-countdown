# 🎉 New Year Countdown App

A modern and responsive New Year Countdown App built using React.js.  
This application displays a live countdown timer showing the remaining days, hours, minutes, and seconds until New Year.

---

## 🚀 Features

- Live Countdown Timer ⏳
- Real-Time Updates Every Second
- Days, Hours, Minutes & Seconds Display
- Responsive Design 📱
- Clean and Modern UI ✨
- New Year Celebration Theme 🎆

---

## 🛠️ Technologies Used

- HTML
- JavaScript
- CSS
